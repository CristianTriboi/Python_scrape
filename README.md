# Python_scrape
You put an array of URLs and it will download and convert them in a single PDF


Check for the version of hombrew with:
brew -v

If you dont have a brew version installed, install it through the cli with:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Check for python with:
pthon3 --v

Check the pip3 installation, it comes usualy with python
pip3 --version

Install the required libraries with:

pip3 install playwright PyPDF2
playwright install
